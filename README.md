## 🚀 Objective
Develop a machine learning model that predicts future sales revenue based on historical business data.  
This project demonstrates how predictive analytics can support **strategic planning, inventory management, and marketing decisions**.

---

## 🛠️ Workflow
1. **Data Preparation**  
   - Load raw dataset (`business_kpi_sample_data.csv`).  
   - Engineer features: Profit, Customer Acquisition Cost (CAC).  
   - Handle missing values and normalize data.

2. **Model Training**  
   - Train a **Linear Regression model** using scikit-learn.  
   - Features: Revenue, Expenses, Customers, Profit, CAC.  
   - Target: Revenue (forecasting future values).

3. **Evaluation**  
   - Metrics:  
     - Mean Absolute Error (MAE)  
     - Root Mean Squared Error (RMSE)  
     - R² Score
